# 금천구 · 서울특별시

> 금천구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-geumcheon.github.io/**

금천구는 조선 시대 금천현(衿川縣)의 이름을 그대로 물려받은 서울 서남권의 자치구로, '비단 내(금천)'라는 지명처럼 안양천이 구의 서쪽을 감싸 흐르는 물의 고장입니다. 옛 시흥군의 관아와 정조가 아버지 능행길에 머물던 시흥행궁이 자리했던 시흥동 일대가 이 지역의 역사적 심장부이며, 오늘날에는 가산·독산의 서울디지털산업단지(G밸리)가 서울 최대 규모의 지식산업 고용거점으로 성장했습니다. 호암산이 남쪽을 병풍처럼 두르고 안양천 벚꽃길이 봄마다 만개하는 등, 도심 산업지대와 자연·유적이 한 구 안에 촘촘히 공존하는 것이 금천구의 개성입니다. 1995년 구로구에서 갈라져 나온 비교적 젊은 행정구이면서도, 880년 수령의 시흥동 은행나무처럼 서울에서 손꼽히는 오랜 뿌리를 함께 품고 있습니다. 좁은 면적에 산업·주거·유적이 밀집해 있어 '작지만 알찬 구'로 불립니다.
